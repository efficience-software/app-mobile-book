Création d'une application mobile

Créer les bases de données

Vous pouvez créer un script pour que les tables soient crées automatiquement à partir de vos modèles dans votre base de données:

Vérifiez que les id de connexion à votre base de données sont corrects dans le fichier server/datasources.json.

Vérifiez que les modèles sont attachés aux bonnes tables dans server/model-config.json.

Lancez la commande:

$ cd server
$ node create-lb-tables.js

Générer les services avec le SDK loopback Angularjs

$ lb-ng ./server/server.js ./client/js/services.js
