Création d'une application mobile

Création de l'application mobile

Créer l'application

$ ionic start <appname> <template>
$ ionic platform add <platform>

Compiler

$ ionic build <platform_name>

Changer les icones et l'écran splash

Placez dans le répertoire resources une image icon.png, icon.psd ou icon.ai et splash.png, splash.png ou splash.ai et exécutez: 

$ ionic resources

Pour ne générer que les icones:

$ ionic resources --icon

Pour ne générer que la splash:

$ ionic resources --splash

Personnaliser la statusbar

Installez le plugin cordova-plugin-statusbar:

$ cordova plugin add cordova-plugin-statusbar

Définissez le style:

$ionicPlatform.ready(function() {
  if (window.StatusBar) {
    // org.apache.cordova.statusbar required
    StatusBar.styleDefault();
  }
});

Récupérer les données de WP

Ajoutez dans votre controller la logique de récupération des données:

.controller('PostsCtrl', function($scope, $http) {

  // You can change this url to experiment with other endpoints
  var postsApi = 'http://url.com/wp-json/posts?_jsonp=JSON_CALLBACK';

  // This should go in a service so we can reuse it
  $http.jsonp( postsApi ).
    success(function(data, status, headers, config) {
      $scope.posts = data;
      console.log( data );
    }).
    error(function(data, status, headers, config) {
      console.log( 'Post load error.' );
    });

})

Et dans votre template:

<ion-view view-title="Posts">
  <ion-content>
    <ion-list>
      <ion-item ng-repeat="post in posts" href="#/app/posts/{{post.ID}}">
        {{post.title}}
      </ion-item>
    </ion-list>
  </ion-content>
</ion-view>
