Création d'une application mobile

Installation sous Mac

Installer node.js et npm

Allez sur https://nodejs.org et téléchargez le dernier package. Suivez les étapes d'installation.

Vérifiez que node est bien installé en lancant la commande:

$ node
> console.log('Hello World');

Vérifiez que npm est bien installé en lancant la commande:

$ npm

Installer Ionic et cordova

$ npm install -g cordova ionic

Vérifiez l'installation:

$ ionic

Créer l'application mobile

Créez l'application:

$ ionic start <appname> <template>
$ ionic platform add <platform>

Pour lancer l'application dans un navigateur web, exécutez:

$ ionic serve

Compilation sous ios

Installer Xcode

Téléchargez et installez XCode sur le site d'apple. https://developer.apple.com/xcode/download/

Compiler sous ios

$ ionic platform add ios
$ ionic build ios

Lancer le simulateur

Installez le simulateur:

$ npm install -g ios-sim

Lancez le simulateur. Utilisez l'otion -l pour que les changements soient pris en compte en temps réel.

$ ionic emulate ios

Compilation sous android

Installer java

http://www.java.com/en/download/mac_download.jsp

Installer le sdk android

Allez en bas de la page http://developer.android.com/sdk/index.html et téléchargez le sdk pour mac osx.

Dé-ziper:

$ unzip ~/<path>/android-sdk_r24.4.1-macosx.zip -d ~/Library/Developer

Copiez une référence de ce chemin à votre profil:

$ echo ~/Library/Developer/android-sdk-macosx | pbcopy

Créez ou modifiez le fichier ~/.bash_profile en ajoutant le chemin copié:

export PATH=${PATH}:/Users/sandrinezecler/Library/Developer/android-sdk-macosx/tools:/Users/sandrinezecler/Library/Developer/android-sdk-macosx/platform-tools

Vérifiez l'installation:

$ android

Installer les dépendances

Installez les dépendances:

  • Android SDK Platform-tools
  • Android SDK Build-tools
  • Sous Android API 22, SDK Platform

Compiler pour android

$ ionic platform add android
$ ionic build android

Installer VirtualBox

Téléchargez sur le site de VirtualBox la version pour OSX:
https://www.virtualbox.org/wiki/Downloads

Installer Genymotion

Créez un compte afin de pouvoir télécharger la version gratuite sur genymotion.com

Lancer un appareil virtuel

Ajoutez un nouvel appareil virtuel en sélectionnant dans la liste "Custom phone - 5.1.0 - API 22 - 768x1280".

Démarrez l'appareil virtuel.

Exécutez:

$ ionic run android
