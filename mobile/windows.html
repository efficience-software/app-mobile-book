Création d'une application mobile

Installation sous windows

Préambule

Pour paramétrer une application mobile sous windows, soit on installe Java, Ant, NodeJS, npm et le SDK Android, puis on configure les chemins:

  • JAVA_HOME
  • ANT_HOME
  • ANDROID_PLATFORM
  • ANDROID_TOOLS

Soit on utilise Visual Studio. On peut préparer une machine Windows facilement en utilisant le package “Tools for Apache Cordova” qu’on peut télécharger pour VS 2013 et qui est inclus dans VS 2015.

Pré-requis

Visual Studio Tools for Apache Cordova

Présent dans VS 2015 ou à télécharger et à installer pour VS 2013.

Vérifiez que la fonctionnalité “Cross Platform Mobile Development” est sélectionné.

Sous “Common Tools”, sélectionnez “Visual Studio Extensibility Tools” et si besoin git et github.

Vérifier l’installation sous une fenêtre d’invite de commandes:

$ node -v
$ cordova -v

Ionic

Installez ionic.

Vérifiez l’installation sous une fenêtre d’invite de commandes:

$ ionic -v

Test sous IOS

Installer les outils IOS

cf. https://msdn.microsoft.com/en-us/library/dn757054(v=vs.120).aspx

  • Mac OS X Mavericks
  • Xcode 6
  • Xcode command-line tools (from Terminal app, use xcode-select --install)
  • Node.js

Il faut également

  • Un compte développeur apple
  • Un ios provisioning profile
  • Une signing identity (cf. Xcode preferences)

Installer le remote agent sous mac

Regardez la doc visual (dépend de la version de visual)

Configurer le remote agent sous visual

Options>Tools for Apache Cordova>Remote Agent Configuration

Lancer le remote agent

vs-mda-remote --secure false

Tester avec un device

Branchez l’Iphone à windows et lancer Itunes.

Emuler un android

Installer Genymotion

Créez un compte afin de pouvoir télécharger la version gratuite sur genymotion.com, suivez les instructions d'installation puis installez virtualbox.

Lancer un appareil virtuel

Ajoutez un nouvel appareil virtuel en sélectionnant dans la liste "Custom phone - 5.1.0 - API 22 - 768x1280".

Démarrez l'appareil virtuel.

Exécutez:

$ ionic run android
