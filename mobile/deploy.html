Création d'une application mobile

Déploiement de l'application

Avec votre téléphone

Branchez votre téléphone et exécutez:

$ ionic run <platform> --device

Avec Ionic Veew

Allez sur ionic.io et créer un compte.
Allez dans le répertoire de votre projet et exécutez les commandes suivantes pour déployer votre application:

$ ionic login
$ ionic upload

Installez Ionic view sur votre téléphone. Vous devriez voir votre application dans la liste d'applications.

Pour partager votre application, exécutez:

$ ionic share <mail>
