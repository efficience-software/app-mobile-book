Installation du plugin WP REST API

Pour récupérer les posts de votre site Wordpress, vous devez installer le plugin WP REST API.

Dans Wordpress, allez dans plugins > Ajouter un nouveau.
Recherchez le plugin WP REST API (WP API).
Cliquez sur installer maintenant pour installer le plugin.
Si l'installation réussit, cliquez sur activer le plugin pour l'activer.

Vérifiez l'installation en ouvrant dans un navigateur:
http://example.com/wp-json

NB: Vous devez utiliser des permaliens agréables à la lecture (pretty permalinks) pour que le plugin puisse fonctionner car il utilise une règle spécifique d'écriture des URLs.
